# Setup Health-Tech Prototype Workflow

This workflow sets up the basic Health-Tech prototype, including the React component and database scripts.

1.  **Initialize the React Project (Optional)**
    If you haven't already, create a new React project.
    // turbo

2.  **Create the HealthTrackerCard Component**
    Create a new file `src/components/HealthTrackerCard.jsx` and copy the provided React component code into it.

3.  **Create Database Scripts Directory**
    Create a `database` folder to store your SQL scripts.
    // turbo

4.  **Save SQL Scripts**
    Create an `init.sql` file in the `database` folder and paste the `CREATE TABLE` and `INSERT/SELECT` statements.

5.  **Run the Application**
    Start your React development server and verify the UI.
